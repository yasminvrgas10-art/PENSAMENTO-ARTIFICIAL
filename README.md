# PENSAMENTO-ARTIFICIAL
 caderno de pesquisas próprio com a utilização do site notebookLM, com o objetivo principal de ter uma IA de estudos e práticar os conhecimentos breves da engenharia de promt para obter respostas práticas

INTRODUÇÃO

O assunto escolhido para o meu "caderno" temático foi John Maynard Keynes, um dos maiores pensadores do século XX, criador da macroeconomia moderna. O objetivo deste projeto foi, aprender de forma prática, a partir de buscas de sites longos, copindo-os e, inserindo no notebookLM  obtendo de maneira rápida a praticidade de obter informações desejadas sobre o assunto, a partir de perguntas 
Essa ferramenta, poderia se dizer parecida ao ChatGPT, entre outros chats artificiais, porém com um toque "especial
